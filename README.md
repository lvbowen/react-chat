# react-chat
聊天 webapp demo（在 06_chat_detail 目录下）

## 项目npm命令
先 `cd 06_chat_detail`
  - `npm run start` 项目本地启动
  - `npm run server` 项目本地服务端启动
  - `npm run build` 项目生产环境编译

## 涉及的技术

* [x] React16
* [x] redux 管理状态
* [x] react-router4 路由
* [x] antd-mobile 蚂蚁金服UI组件库
* [x] express 
* [x] mongodb
* [x] socket.io
